MPack — Build & Install

Files included:
 - pom.xml
 - src/main/resources/plugin.yml
 - src/main/resources/config.yml
 - src/main/java/com/samziri/mpack/MPack.java

Build steps (Maven):
 1. Put the project in a folder with the above structure.
 2. Run: mvn clean package
 3. The shaded JAR will be in target/mpack-1.0.0.jar
 4. Drop the JAR into your server's plugins/ folder and restart or reload.

Notes:
 - Edit the POM  to match your server's API (1.16.5, 1.20.2, etc).
 - Add the rest of the Java classes (commands/managers) to implement features.
 - plugin.yml already declares many commands; implement their executors in code.
